# OptiManage

## To View a Development Preview

The server and dependencies for the front-end and back-end must be installed and run separately, using two terminals.

For the back end:

```
  > cd backend;
  \backend> npm i;
  \backend> npm start;
```

For the front end:

```
  > npm i;
  > npm run dev;
```
